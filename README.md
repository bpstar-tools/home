# スタレゾ ツール置き場

スターレゾナンス（Star Resonance / BPSR）用の自作ツール・図鑑をまとめたサイトです。

`index.html` がトップページ（一覧）で、各ツールへ相対リンクで飛びます。GitHub Pages を有効にすると、そのまま公開できます。

## 収録ページ

| ファイル | 内容 |
|---|---|
| `index.html` | トップページ（ツール一覧） |
| `タクティクスボード.html` | ギミックの説明・散会図の作成 |
| `バトルイマジン図鑑.html` | 凸毎の効果、英語・中国語名の確認 |
| `海武器素材計算機.html` | 完成日の算出・必要購入数のシミュ |
| `精錬素材シミュレーター.html` | 素材を自作するか買うかの判断 |
| `翻訳拡張機能.html` | Maxroll翻訳ツール（ブラウザ拡張）の解説とDL |
| `ZDPS日本語化パッチ.html` | DPSメーター ZDPS の日本語化パッチの解説とDL |

## 公開方法（GitHub Pages）

リポジトリの Settings → Pages → Branch を `main` / `/(root)` に設定すると公開されます。
