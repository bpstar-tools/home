スタレゾMaxroll 英語 → 日本語 翻訳ツール

「スターレゾナンス（Star Resonance / BPSR）」の Maxroll の Web ページを日本語化するブラウザ拡張です（英語→日本語専用）。

ゲーム用語（公式名）と説明文を、専用の辞書で正確な日本語に置換します。

全文翻訳は翻訳エンジンを選んで使用（Chrome内蔵翻訳＝無料 / DeepL＝高品質）。どちらも用語は辞書どおりに固定されます。

maxroll.gg では自動発動。その他の英語サイトでも手動（ボタン / Alt+T / 右クリック）で使えます。

ダウンロード

Chrome / Edge 版 bpsr-translate-chrome.zip（約 62 KB） Firefox 版 bpsr-translate-firefox.xpi（約 62 KB）

インストール方法

  1. Chrome / Edge 版（zip）をダウンロードして解凍します。
  2. chrome://extensions を開きます。
  3. 右上の「デベロッパーモード」を ON にします。
  4. 「パッケージ化されていない拡張機能を読み込む」→ 解凍した bpsr-translate-chrome フォルダを選択します。
  5. ツールバーにアイコンが出れば完了です。
解凍フォルダは消さないでください。更新時は新しい zip を解凍し、拡張ページで リロード（↻） してください。
  1. Chrome / Edge 版（zip）をダウンロードして解凍します。
  2. edge://extensions を開きます。
  3. 左下の「開発者モード」を ON にします。
  4. 「展開して読み込み」→ 解凍した bpsr-translate-chrome フォルダを選択します。
  5. ツールバーにアイコンが出れば完了です。
解凍フォルダは消さないでください。更新時は新しい zip を解凍し、拡張ページで リロード（↻） してください。
Edge は内蔵翻訳APIが使えないため、全文翻訳は DeepL 推奨 です。

通常版 Firefox は未署名拡張を常用できないため、Developer Edition / Nightly を前提とします。

  1. about:config を開き、xpinstall.signatures.required を false に設定します（最初の一度だけ）。
  2. Firefox 版（.xpi）をダウンロードします。
  3. about:addons → 歯車 ⚙ →「ファイルからアドオンをインストール」→ .xpi を選択します。
Firefox は内蔵翻訳APIが無いため、全文翻訳は DeepL 推奨 です。

初期設定

拡張アイコンをクリックして設定します。

翻訳エンジンChrome内蔵 / DeepL / 辞書のみ（既定は Chrome内蔵）
DeepL APIキーDeepL を選んだ時のみ必要（取得方法は後述）。
動作モード「手動」=ボタン / Alt+T で発動。「常に翻訳」=開くと自動。
自動発動サイト記載サイトは開くと自動発動（既定 maxroll.gg）。
手動で翻訳する3つの方法：ポップアップの「▶ このページを今すぐ翻訳」／ Alt+T ／ 右クリック →「このページを翻訳」。

Chrome内蔵翻訳を使う（無料）

  1. ポップアップで翻訳エンジン「Chrome内蔵翻訳」を選んで保存します（既定で選択済み）。
  2. maxroll のページを開きます。
  3. 初回のみ、画面上部に「▶ クリックでChrome翻訳を有効化（初回のみモデルDL）」のバナーが出るのでクリックします。
    • クリックで翻訳モデルがダウンロードされます（数十MB・一度きり）。完了後そのまま全文翻訳されます。
  4. 2回目以降は自動で全文翻訳されます。
Chrome 138未満や Edge / Firefox では内蔵翻訳が使えず、自動で「辞書のみ」になります。その場合は DeepL を使ってください。

DeepL を使う（高品質・推奨：Edge / Firefox）

辞書に無い文章も DeepL が翻訳し、ゲーム用語は辞書どおりに固定されます。

DeepL 無料APIキーの取得手順

  1. DeepL API（開発者向け） を開き、アカウントを作成します（既にDeepLアカウントがあればログイン）。
  2. プランは 「DeepL API Free」 を選択します（月50万文字まで無料）。
    • 登録時に本人確認のためクレジットカード情報を求められる場合がありますが、無料プランは課金されません（上限超過時も自動課金されず停止します）。
  3. ログイン後、アカウントのAPIキー画面 を開きます。
  4. 表示される Authentication Key（認証キー）をコピーします。
    • 無料プランのキーは末尾が :fx（例: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx:fx）。
  5. 拡張ポップアップで翻訳エンジン「DeepL」を選び、DeepL APIキー欄に貼り付けて「保存して再読み込み」します。
